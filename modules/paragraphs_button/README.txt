INTRODUCTION
------------

Paragraphs Button allows you choose "Add new" button postion on Top or Bottom 
(default) when you use multi-values with Paragraph field.

Only  work with "Add mode" is Button type


REQUIREMENTS
------------

Paragraphs module


INSTALLATION
------------

Using the module is fairly simple just enable it and you're good to go.


CONFIGURATION
-------------

Go to Paragraph field setting for content type then choose "Top"  or "Bottom", 
only  work with "Add mode" is Button type. Then in edit content form , you will 
see button change position on top or bottom of Table fields.
